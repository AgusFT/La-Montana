# Styles

## Objetivo

Centralizar la organización de estilos globales del frontend.

---

## Estructura sugerida

```txt
styles/
│
├── tokens.css
├── layout.css
├── components.css
├── dashboard.css
└── login.css
```

---

## Responsabilidades

### tokens.css

Variables globales:

- colores
- tipografías
- espaciados
- radios
- sombras

---

### layout.css

Estructuras generales:

- contenedores
- grids
- layouts

---

### components.css

Estilos compartidos por componentes reutilizables.

---

### dashboard.css

Estilos específicos del dashboard.

---

### login.css

Estilos específicos de autenticación.

---

## Objetivo final

Mantener estilos organizados, escalables y fáciles de mantener.
