# Types

## Objetivo

Centralizar contratos TypeScript compartidos por toda la aplicación.

---

## Ejemplos

```txt
types/
│
├── pedido.ts
├── usuario.ts
├── archivo.ts
├── auth.ts
└── cotizacion.ts
```

---

## Beneficios

- tipado consistente
- reutilización
- reducción de errores

---

## Recomendación

Definir interfaces y tipos reutilizables.

Evitar duplicar tipos en múltiples módulos.
