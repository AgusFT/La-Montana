# Mocks

## Objetivo

Contener datos simulados utilizados durante el desarrollo y prototipado.

---

## Ejemplos

```txt
mocks/
│
├── pedidos.ts
├── usuarios.ts
└── dashboard.ts
```

---

## Uso

Permiten construir interfaces antes de disponer de un backend funcional.

---

## Importante

Los mocks no deben utilizarse en producción.

Toda la información productiva debe provenir de Supabase.
