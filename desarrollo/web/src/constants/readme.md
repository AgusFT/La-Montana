# Constants

## Objetivo

Centralizar valores constantes utilizados por la aplicación.

---

## Ejemplos

```txt
constants/
│
├── roles.ts
├── estadosInternos.ts
├── estadosVisibles.ts
└── estadosFinancieros.ts
```

---

## Casos de uso

- roles del sistema
- estados de pedidos
- configuraciones globales
- valores estáticos

---

## Beneficios

Evita valores hardcodeados distribuidos por el proyecto.
